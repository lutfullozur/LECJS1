# _<span style='color:red'>DOM</span>_
# _<span>DOM (Document Object Model) – это объектная модель документа, которую браузер создает в памяти компьютера на основании HTML-кода, полученного им от сервера. Иными словами, это представление HTML-документа в виде дерева тегов. DOM обеспечивает представление документа в виде структурированной группы узлов и объектов, которые имеют свойства и методы <span>_
# _<span style='color:red'>BOM</span>_
# _<span>BOM (Browser Object Model) в JavaScript представляет собой набор объектов, предоставляемых браузерной средой. Он позволяет JavaScript взаимодействовать с самим браузером, обрабатывать свойства окна и управлять взаимодействием с пользователем</span>_
# _<span style='color:yellow'>Вот некоторые ключевые компоненты BOM</span>_
# _<span style ='color:red'>1</span>_ _<span>- Объект window: Это корневой объект BOM, представляющий окно браузера. Он служит глобальным объектом для JavaScript в среде веб-браузера </span>_

# _<span style ='color:red'>2</span>_ _<span>- Объект document: Хотя он также является частью DOM, он относится и к BOM. Объект document представляет веб-страницу и предоставляет методы для работы с её содержимым </span>_
![alt text](image-1.png)
# _<span style ='color:red'>3</span>_ _<span>- Объект navigator: Предоставляет информацию о браузере, такую как его название, версия и платформа </span>_
![alt text](image-2.png)
# _<span style ='color:red'>4</span>_ _<span>- Объект history: Позволяет манипулировать историей сессии браузера, управляя навигацией в пределах сессииа </span>_
![alt text](image-3.png)
# _<span style ='color:red'>5</span>_ _<span>- Объект screen: Предоставляет информацию об экране пользователя, такую как его ширина, высота и глубина цвета.а </span>_
![alt text](image-4.png)
# _<span>BOM позволяет JavaScript взаимодействовать с браузером и управлять различными аспектами веб-приложений. Это мощный инструмент для создания интерактивных веб-сайтов и приложений</span>_

# _<span style='color:red'>Особенности DOM</span>_
# _<span style='color:red'>1 </span>_ _<span>-  JavaScript может изменять все элементы HTML на странице.</span>_
# _<span style='color:red'>2 </span>_ _<span>-  JavaScript может изменять все атрибуты HTML на странице. </span>_
# _<span style='color:red'>3 </span>_ _<span>-  JavaScript может изменить все стили CSS на странице.</span>_
# _<span style='color:red'>4 </span>_ _<span>- JavaScript может удалять существующие элементы и атрибуты HTML. </span>_
# _<span style='color:red'>5 </span>_ _<span>- JavaScript может добавлять новые элементы и атрибуты HTML. </span>_
# _<span style='color:red'>6 </span>_ _<span>- JavaScript может реагировать на все существующие HTML-события на странице </span>_
# _<span style='color:red'>7 </span>_ _<span>-  JavaScript может запускать новые события HTML на странице </span>_
# _<span style='color:red'>Метод querySelector </span>_ _<span style='color:'> - возвращает первый дочерний элемент, который соответствует указанному селектору(ам) CSS элемента. Метод запроса Selector All можно использовать для доступа ко всем элементам, которые соответствуют указанному селектору CSS</span>_
# _<span style='color:red'> innerHTML </span>_ _<span style='color:'> - это свойство полностью обеспечивает простой способ заменить элементарный элемент. Например, все требования тело элемента можно удалить</span>_

# _<span style='color:red'> onclick </span>_ _<span style='color:'> - Событие onclick в JavaScript активируется, когда пользователь кликает на определенный элемент страницы, такой как кнопка, ссылка, изображение или любой другой элемент, на который можно нажать</span>_
